# To-Do List Application

A simple GUI-based task management application built with Python and `breezypythongui`.

## Features
- **Add tasks**: Enter tasks in the input field and click "Add"
- **Mark tasks complete/incomplete**: Single-click to toggle (green = complete)
- **Delete tasks**: Double-click a task and confirm to delete
- **Clear all tasks**: Remove all tasks with confirmation
- **Input validation**: Prevents adding empty tasks
- **Exit Application**: Stops the program upon exiting

## Requirements
- Python 3.x
- `breezypythongui` library
- `tkinter` (usually included with Python)

## Installation
1. Ensure Python is installed on your system
